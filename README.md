# HR Attendance Dashboard – Excel Project

## Overview  
This is an Excel-based HR Attendance Dashboard that analyzes employee attendance data using formulas, pivot tables and charts. It helps visualize department-wise attendance and overall attendance summary.

## Project Contents  
- `Pratik_Excel_DataAnalyst.xlsx` — The complete workbook containing raw data, formulas, pivot tables and dashboard.  
- (Optional) Screenshots folder — to view dashboard without opening Excel.

## Features & Skills Demonstrated  
- Excel formulas: `SUMIFS`, `COUNTIFS`, `IF`, `XLOOKUP`, `INDEX-MATCH`, etc.  
- Data cleaning (handling missing values, text standardization)  
- Pivot Tables & Pivot Charts for aggregated view  
- Interactive dashboard with KPIs and charts  
- Clean layout for easy readability  

## How to Use  
1. Download or clone the repository.  
2. Open `Pratik_Excel_DataAnalyst.xlsx` in Microsoft Excel or similar.  
3. Navigate between sheets: raw data, pivot, dashboard.  
4. View attendance summary and dashboards.  

## Why This Project Matters  
Many companies manage attendance and HR data using Excel. This dashboard demonstrates ability to clean data, derive meaningful insights and present them professionally — skills expected for a Data-Analyst / MIS-Analyst role.
